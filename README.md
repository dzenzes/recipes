# Daniels Rezepte

Seite um Rezepte, die ich gerne koche, zu verwalten.

Erstellt mit [11ty](https://www.11ty.dev) und [cooklang](https://cooklang.org)

## Lokales Setup

```
npm install
```

## Seite starten

```
npm start
```

## Seite bauen

Die Seite wird im `_site` Verzeichnis gebaut.

```
npm run build
```
